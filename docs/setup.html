Magical Girls Gallery Setup

This page allows to import M-Girls into the Gallery.

Click anywhere, or drag&drop your gallery data file. It can be found in /data/data/com.aniplex.magireco.en/files/, or generated with NAgitan from backup archive.

If you don't have the file, try to erase AP counter and type something to get predefined example set. For example, mackingham.
